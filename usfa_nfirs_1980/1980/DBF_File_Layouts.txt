NFIRS 4.1 DBF FILE TABLE STRUCTURES/LAYOUTS

The following Tables make up the various record types in the Incident and the
Equipment Report files in NFIRS 4.1. (Please note the the 05 record type is not
included here). Includes the address extract from the master file 10 record.
--------------------------------------------------------------------------------
